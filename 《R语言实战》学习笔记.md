R语句由函数和赋值构成。 *R使用<-而非传统的=来赋值*
<br>`x <- rnorm(5)` 或者`rnorm(5)->x`皆可。
<br>注释用`#`开头。

* 用`c`可以将数据以向量的形式输入

```
> age <- c(1,3,5,2,11,9,3,9,12,3)
> weight <- c(4.4,5.3,7.2,5.2,8.5,7.3,6.0,10.4,10.2,6.1)

> mean(weight)
[1] 7.06
> sd(weight)
[1] 2.077498
> cor(age,weight)
[1] 0.9075655
> plot(age,weight)
```
<br> ∴ 这组婴儿的平均体重是7.06千克；标准差是2.08千克；月龄和体重存在较强的线性关系。

